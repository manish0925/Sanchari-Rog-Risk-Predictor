Sanchari Rog Risk Predictor 🏥
The Sanchari Rog Risk Predictor is a Flask-based web application designed to help users assess risk factors for communicable diseases. It features a machine-learning-ready backend and a functional Contact Us form that sends automated email notifications to the administrator.

🚀 Features
Disease Risk Prediction: Analyzes user input to provide risk assessments for various communicable diseases.

Automated Contact Form: Integrated with Flask-Mail to send user queries directly to your Gmail inbox.

Secure Configuration: Uses environment variables (.env) to protect sensitive credentials like email passwords.

Responsive UI: Built with Bootstrap 4 to ensure the application works perfectly on desktops, tablets, and mobile phones.

🛠️ Tech Stack
Backend: Python (Flask)

Frontend: HTML5, CSS3, Bootstrap 4

Email Service: Flask-Mail (SMTP via Gmail)
